### Running the project folder
```
$ conda activate ./env
$ jupyter notebook
```

### Update packages
```
$ pip install -U package-name
```

### Exporting Enviroment
```
$ conda export > filename.yml
```

### Using the Exported Environment
```
$ conda env create --prefix ./env -f ./environment.yml
```
